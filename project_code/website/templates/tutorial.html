GUIDE

This website shows the results of our text mining application, these results can be visualised in 2 different diagrams. The first is a sunburst diagram which consists of three connected circles, the first circle contains the organisms, the second circle the stress conditions found per organism and the third circle the genes found per stress condition. The second diagram is a tree where again the first node shows the organisms found, this then branches out to show what stress conditions were found for this organism and again the tree branches out from the stress conditions to show per condition what genes are involved here.
For the sunburst diagram you can click on a specific item (organism, condition or gene) and a table will show you some more specific information about this item.
