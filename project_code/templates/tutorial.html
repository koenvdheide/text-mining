GUIDE

This website shows the results of our text mining application. If you head over to the Results tab, you will be able to select a way of visualisation.
The results can either be visualised in a sunburst diagram or a tree. More specification can be read below.

Visualisation can be displayed in 2 ways.

Option 1: Sunburst diagram
This diagram consists of three connected layers of circles.
Layer 1, as shown in the red circle on the right, displays the organisms having Anthocyanins.
Layer 2, as shown in the black circle on the right, displays the stress conditions per organism.
Layer 3, as shown in the yellow circle on the right, displays the genes found per stress condition.
*The circles are indicative and not part of actual sunburst

You can click an entity (organism, condition or gene) and the sunburst will zoom in, showing more specific information about this item.

Option 2: Tree diagram
This diagram is a tree (see image below) where again the first node shows the organisms found, this then branches out to show what stress conditions were found for this organism and again the tree branches out from the stress conditions to show per condition what genes are involved here.
